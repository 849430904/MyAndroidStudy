ALGroup.m

Overview

ALGroup defines a group of ALDevice under a specified name. Currently groups are stored as AylaDatum. Behind the scenes ALGroup uses a dictionary stored in an AylaDatum object to serve as index to allow for random access to the groups. each group name should is unique and used as means to identify it.

Parameters

Discussion

Executes a given block using each object in the array, starting with the first object and continuing through the array to the last object.

The block takes three arguments: - devcice The device in the source. - idx The index of the element in the source. - stop A reference to a Boolean value. The block can set the value to YES to stop further processing of the array. The stop argument is an out-only argument. You should only ever set this Boolean to YES within the block.
