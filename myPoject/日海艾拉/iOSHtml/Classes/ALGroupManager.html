Inherits from NSObject
Declared in ALGroupManager.h
ALGroupManager.m

Overview

Retrieves, creates and deletes ALGroup instances, manages the group index. To update the members of the group use the ALGroup instance itself.

Group names are unique, if you create a group with a taken name the method will return an ALGroupManagerErrorGroupNameExists error.
