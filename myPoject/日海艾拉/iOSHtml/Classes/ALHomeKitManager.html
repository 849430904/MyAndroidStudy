ALHomeKitManager.m

Overview

Home Kit manager for tracking Home Kit flows.

Home Kit manager has to be initialized before any Home Kit related calls. (All other home kit related apis will return nil/NO if Home kit manager can not be found)

Note: Home Kit Manager is following singleton design.
Use startWithHomeKitParameters: to initialize a Home Kit manager. And use +currentHomeKitManager: to access to running Home Kit manager.
After the first time call of startWithHomeKitParameters:, this api will behavior similarly as +currentHomeKitManager: .
