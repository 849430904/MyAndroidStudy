Overview

Class to add or edit a batch action. Shows batch action name and device actions not currently associated with batch action object, to add from.

Other Methods

– setBatchAction:associatedActions:

Sets initial batch action and all associated actions when editing existing batch action object. View will only show actions not already associated with this batch action object. User can add from those actions.

- (void)setBatchAction:(ALBatchAction *)batchAction associatedActions:(NSArray<ALAction*> *)actions

Parameters

batchAction

ALBatchAction object to be updated
actions

Array of ALAction objects. This array of ALAction objects (already fetched) is set to optimize UI and avoid an extra network call
