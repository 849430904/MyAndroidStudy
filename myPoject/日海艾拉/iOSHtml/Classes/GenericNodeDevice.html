– friendlyNameForProperty:

Optional Override. Returns a friendly name for the specified property name, you can override this method to provide a name suitable for the end user for instance this name will be shown when the user sets a notification up.

- (NSString *)friendlyNameForProperty:(AylaProperty *)property

Parameters

property

The name of the property of which the friendly name will be returned.

Return Value

The friendly name of the specified property.

Discussion

Optional Override. Returns a friendly name for the specified property name, you can override this method to provide a name suitable for the end user for instance this name will be shown when the user sets a notification up.

The default behavior is to return the displayName first, if no display name has been provided, the method returns the name property in the AylaProperty.
