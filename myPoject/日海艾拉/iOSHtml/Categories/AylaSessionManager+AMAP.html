AylaSessionManager(AMAP) Category Reference

Declared in ALSessionManager.h
ALSessionManager.m

Overview

The AMAP category adds functionality to AylaSessionManager, this category provides a layer of synchronization among managers. You should use the amap_ prefixed methods whenever there’s a choice between original and amap versions to keep data in synch.
