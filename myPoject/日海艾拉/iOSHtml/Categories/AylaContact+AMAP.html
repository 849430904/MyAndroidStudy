Overview

This category a contact which can be used with a Trigger App.

Creation

You should create instances of this class using the createContact instance in the ALContactManager of the session.

ALContact * contact = [contactManager createContact];

After the instance has been created, you can modify its properties, the changes will be persisted in the cloud after you call [ALContactManager saveContact:success:failure:]

contact.firstName = @"John";
contact.lastName = @"Doe";
[contactManager saveContact:contact success:success failure:failure];

If you write or modify the properties after the creation, you should call saveContact:success:failure: again to persist the changes in the cloud.

contact.email = @"jdoe@abcde.com";
contact.countryCode = @"1";
contact.phoneNumber = @"9990001111";
