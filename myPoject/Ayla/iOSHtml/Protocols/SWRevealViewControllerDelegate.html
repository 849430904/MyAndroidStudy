– revealController:panGestureBeganFromLocation:progress:overProgress:

The following methods provide a means to track the evolution of the gesture recognizer. The ‘location’ parameter is the X origin coordinate of the front view as the user drags it The ‘progress’ parameter is a number ranging from 0 to 1 indicating the front view location relative to the rearRevealWidth or rightRevealWidth. 1 is fully revealed, dragging ocurring in the overDraw region will result in values above 1. The ‘overProgress’ parameter is a number ranging from 0 to 1 indicating the front view location relative to the overdraw region. 0 is fully revealed, 1 is fully overdrawn. Negative values occur inside the normal reveal region

- (void)revealController:(SWRevealViewController *)revealController panGestureBeganFromLocation:(CGFloat)location progress:(CGFloat)progress overProgress:(CGFloat)overProgress
