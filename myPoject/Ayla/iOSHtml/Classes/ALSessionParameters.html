ALSessionParameters.m

Overview

ALSessionParameters contains the parameters to configure the session, the default init method will initialize the instance with the default ALAccountCreator and ALDeviceCreator classes.

If you have custom creator classes they can be overriden with the accountCreator and deviceCreator properties.

You must assign the values of all the properties.
