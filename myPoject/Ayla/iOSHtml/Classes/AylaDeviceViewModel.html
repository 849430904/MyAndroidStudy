ALDevice+DeviceViews.m

Overview

Declares the recommended methods to override, subclassing allows you to specify custom views for different devices. Override the methods in the subclasses of AylaDeviceViewModel to add new views for each function.

Home Kit interface for ALDevice objects

Note: Requires a running ALHomeKitManager.

These methods are final, it’s not adviced to override them
