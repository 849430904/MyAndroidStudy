Overview

A Property trigger defines a condition under which some actions (Trigger Apps) can be performed to notify the contacts.

Parameters

The name of the trigger.

value

The value to compare to.

Discussion

Use this method to configure the trigger with a specific value and a comparision type which can be one from AylaPropertyTriggerCompare:


    [propertyTrigger configureWithName:@"TurnOn" triggerWhenPropertyIs:AylaPropertyTriggerCompareEqualTo value:@"1"];

Discussion
