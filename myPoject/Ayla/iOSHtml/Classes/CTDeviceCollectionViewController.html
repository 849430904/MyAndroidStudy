CTDeviceCollectionViewController.m

Overview

Base class for controllers displaying a list of devices.

The class must be provided with a deviceSource which will provide the elements to display. The session manager will come in handy when the controller needs to pass information to a detail view.

The controller listens to changes from devices and device manager.

You can stop receiving device updates by calling the stopDeviceUpdates method.
